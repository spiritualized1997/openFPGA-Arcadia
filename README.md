# openFPGA-Arcadia
Arcadia core for openFPGA on Analogue Pocket
-

Ltrig + up = 0  
Ltrig + right = 1  
Ltrig + down = 2  
Ltrig + left = 3  
Rtrig + up = 4  
Rtrig + right = 5  
Rtrig + down = 6  
Rtrig + left = 7  

Ltrig + Rtrig + up = 8  
Ltrig + Rtrig + right = 9  
Ltrig + Rtrig + down = *  

These buttons don't require chording:

X = #  
A = option  
start = start  
select = select  
Y = fire button
